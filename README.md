# Платформа для построения графиков,Анализа спектра и рабочих зон режущих инструментов

## Цель программы

Inspectrum - программа, позволяющая комплексно анализировать вибрационные данные оборудования для диагностики состояния механических систем,выявления аномалий в вибрационных характеристиках и автоматизации обработки больших массивов экспериментальных данных

- **Тип программы:** Прикладная  
- **Платформы:** Windows 10, 11  
- **Язык интерфейса:** Русский

## Назначение программы

### Пути программы 

~\Массивы данных\~

~\RMS  (внутри этого файла ищет папки X,Y,Z формата .mat)

~\Виброускорение  (внутри этого файла ищет папки Ax,Ay,Az формата .mat)

~\Виброскорость  (внутри этого файла ищет папки X,Y,Z формата .mat)

~\Виброперемещение  (внутри этого файла ищет папки X,Y,Z формата .mat)

### 1. Анализ вибрационных параметров инструмента

- Анализ виброускорения 
- Анализ виброперемещения 
- Анализ виброскорости 
- Анализ временных интервалов виброперемещения 
- Анализ спектра виброускорения  по всему графику
- Отображение статистики обнаруженных интервалов
- Детальный анализ интервалов и их спектров

### 2. Диагностика оборудования

- Обнаружение неисправностей по характерным частотам
- Оценка износа подшипников и других компонентов
- Контроль балансировки вращающихся механизмов

### 3. Исследование динамических процессов

- Анализ переходных процессов при пуске/остановке
- Выявление резонансных частот системы
- Изучение периодичности рабочих циклов

## Основные функции

### 1. Загрузка и подготовка данных

- Автоматическое сканирование структурированных директорий
- Чтение данных из MATLAB (.mat)
- Автоматическое определение типа данных по имени папки
- Очистка данных и создание временной оси
- Центрирование данных относительно нуля

### 2. Визуализация временных рядов

- Синхронизированные графики для трех типов вибрационных данных
- Интерактивное масштабирование и панорамирование
- Выбор отображения: все графики вместе или по отдельности
- Цветовая дифференциация данных и интервалов
- Отображение порогов шума и статистических меток

### 3. Анализ временных интервалов

- Автоматическое детектирование активных интервалов на основе статистических порогов
- Расчет характеристик интервалов:
  - Временные границы и длительность
  - Амплитудные характеристики (среднее, СКО, пиковые значения)
  - Количество и частота пиков внутри интервала
- Настраиваемые параметры обнаружения

### 4. Спектральный анализ

- Быстрое преобразование Фурье (БПФ) для всего сигнала и отдельных интервалов
- Выявление доминирующих частот и гармоник
- Автоматическая маркировка основных частотных компонент
- Анализ спектральной плотности мощности

### 5. Диагностические функции
- Обнаружение неисправностей по характерным частотам
- Оценка износа подшипников (анализ частот качения элементов)
- Контроль балансировки вращающихся механизмов
- Выявление резонансных частот системы
- Анализ переходных процессов при пуске/остановке

### 6. Статистический анализ

- Расчет базовых статистик: среднее, минимум/максимум, СКО, размах
- Подсчет количества активных интервалов и общей длительности активности
- Генерация сводных отчетов по анализу
- Экспорт статистики в CSV формате

### 7. Детальный анализ интервалов

- Отдельные окна для каждого обнаруженного интервала
- Двойное представление: временной ряд + амплитудный спектр
- Возможность сравнения интервалов между собой
- Экспорт данных конкретного интервала

### 8. Настраиваемые параметры

Программа предоставляет гибкие настройки под конкретные задачи:
- **Порог обнаружения шума:** 0.05 (по умолчанию)
- **Минимальная длительность интервала:** 10 отсчетов
- **Размер окна сглаживания:** 3
- **Частота дискретизации:** 3243 Гц (настраивается под ваши данные)
- **Пороги амплитуд:** настройка уровней для детектирования аномалий

### 9. Интерактивные элементы управления

- Выбор оси: X, Y, Z
- Выбор типа отображаемых графиков
- Инструменты масштабирования и панорамирования
- Кнопки быстрого доступа к функциям

## Преимущества

### 1. Автоматизация и удобство использования 

- Автоматическое обнаружение файлов в структурированных директориях
- Синхронизация выборки файлов по именам для разных осей и типов данных
- Поддержка форматов .mat, .csv с автоматическим определением типа файла

### 2. Комплексный анализ данных

- Трехосный анализ (X, Y, Z) вибрационных параметров
- Три типа вибрационных характеристик в одном интерфейсе
- Синхронизированное отображение временных рядов

### 3. Продвинутые алгоритмы обработки

- Алгоритм обнаружения активных интервалов на основе статистических порогов
- Автоматическое определение шума с настраиваемым порогом
- Сглаживание данных скользящим средним
- Спектральный анализ (БПФ) для выявления доминирующих частот

### 4. Гибкая визуализация

- Интерактивные графики с панорамированием и масштабированием
- Настраиваемое отображение: все графики вместе или по отдельности
- Цветовая дифференциация данных и интервалов
- Аннотации и легенды с ключевыми метриками

### 5. Экспорт и сохранение

- **Графики:** сохранение в PNG, PDF, SVG, JPEG с настройкой качества и разрешения
- **Данные:** экспорт временных рядов и спектров в CSV
- **Статистика:** автоматическая генерация отчетов в текстовом формате
- **Автоматизация:** поддержка пакетного режима для обработки серий измерений
